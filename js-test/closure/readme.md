The closure examples in Javascript:The Definitive Guide

closure闭包，从字面上可以理解为，函数定义时将变量“包裹”了起来。

说到这里，就不得不说一下javascript的作用域了。
js不支持块级作用域，而是函数作用域。这个作用域是词法划分的（lexical scoping），而不是动态划分的。换句话来说，js作用域在函数定义时决定，而不是函数执行时决定的。为了实现这种词法作用域，函数对象需要引用当前作用域链（scope chain）。
函数体内部的变量都被保存在函数作用域中，不同函数对象可以用一个跨越不同执行环境的作用域链关联起来，便可以实现函数引用其定义时变量的机制了。

读到这里，你大概就可以真正的理解闭包了。
其实闭包很简单，就是“函数定义时的作用域到函数执行时依然有效”。（看起来好像是函数定义时将变量“包裹”了起来，不管外界怎样变化，“包裹”都在）

而这种机制也导致一些变量的空间不能够被释放，这是后话了。
